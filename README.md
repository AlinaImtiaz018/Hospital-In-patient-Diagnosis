# Hospital in-Patient Diagnosis Project

In this project, we conduct an Exploratory Data Analysis (EDA) on hospital in-patients dataset focusing on the diseases they are diagnosed with initially from the year 2018 to 2022. The data is divided into different age groups spanning from 0 to 90 years, consisting the information about the average length of stay at the hospital. 


## Dataset Citation

The dataset used in this project was obtained from the [Destatis Website](https://www-genesis.destatis.de/genesis/). 

Destatis. (2024). Hospital patients: Germany, years, gender, age groups, place of residence of the patient, main diagnosis ICD-10 (1-3 digit hierarchy). Retrieved from [URL](https://www-genesis.destatis.de/genesis/online?operation=sprachwechsel&language=en).
